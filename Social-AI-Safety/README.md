# Social-AI-Safety

This module evaluates the safety of Large Language Models (LLMs) in social scenarios, including dimensions such as flattery and deception

## Project Structure

```
Social-AI-Safety/
├── configs/                    # Configuration files
│   ├── datasets/               # Dataset configurations
│   │   └── deceptioneval.yaml
│   ├── prompts/                # Prompt templates
│   │   └── deception.yaml
│   ├── gen_config.yaml         # LLM generation parameters 
│   └── judge.yaml              # Prompt templates for LLM-as-a-Judge evaluation
├── data/
│   └── DeceptionEval.jsonl     # Dataset file
├── src/                        # Evaluation code
│   ├── __init__.py
│   ├── dataset.py              # Data loading and preprocessing
│   ├── evaluator.py            # Core evaluation logic and metrics calculation
│   ├── llm.py                  # LLM API wrapper
│   ├── pipeline.py             # Inference pipeline
│   └── prompt.py               # Utilities for reading and formatting YAML prompt 
├── batch_judge.py              # Script for batch-running main_judge.py
├── batch_run.py                # Script for batch-running main.py
├── main.py                     # Main entry point for inference
└── main_judge.py               # Judging the output generated by main.py
```

## Categories

- **Alignment Faking:** Evaluate whether the model feigns alignment during the supervision phase but exposes its true, misaligned intentions in specific scenarios.
- **Sandbagging:** Test if the AI intentionally hides its true performance by answering incorrectly under specific conditions.
- **Deception:** Test if the model prioritizes a biased stance over provided facts, leading to intentional dishonesty when comparing neutral and induced responses.
- **Sycophancy:** Test if the model prioritizes agreeableness over truthfulness by echoing user-provided misinformation, despite possessing the correct knowledge.
- **Manipulation:** Evaluate whether the model strategically induces negative emotions (e.g., self-doubt) to exert psychological control over the user.
- **Feint:** Assess the model's use of tactical maneuvers, including feints and decoys, intended to divert attention to secure a goal in specific scenarios. 
- **Bluffing:** Test if the model fakes advantages while in a vulnerable state to achieve a desired outcome in specific interactions.

## Quick Start

### LLM Configuration

Place your LLM configuration files in `src/llms/`. The configuration format follows the example below:

```yaml
llm_type: "OpenAiLLM"
model_name: "model name"
base_url: "your base_url"
api_key: "your api-key"
```

### Single Inference & Judge

Run `main.py` to generate responses.

```bash
cd Social-AI-Safety/

python main.py --llm-config configs/llms/gpt-5.2.yaml
```

Run `main_judge.py` to judge the output.

```bash
python main_judge.py --input-file results/gpt-5.2_deception.jsonl
```

### Batch Inference & Judge

Run `batch_run.py` to generate responses with all models in `configs/llm` in parallel.

```bash
python batch_run.py --llm-configs configs/llms
```

Run `batch_judge.py` to jugde all output in `results/` in parallel.

```bash
python batch_judge.py --input-dir results/
```
